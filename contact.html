Work

gesof/maker-bundle

As symfony programmers, starting with the earliest versions, we have always appreciated the way this framework has evolved ... until version 4. Then a new way of organizing sources was proposed by giving up the "bundle concept", which for us would have meant giving up to millions of lines of code already written. Since there is no longer native support for creating bundles, we had to rebuild a generator command that would allow the continued use of a structure that allows the separation of logic into modules and their reuse. And so MakerBundle was born

Learn more

gesof/user-bundle

We are now at version 5 of Symfony Framework and we had to update one of the platforms made on symfony 4 which we postponed because one of the key packages is no longer developed, namely FOSUserBundle, the package that provides a registration system and authentication so necessary for any platform. So we had to invest time in creating a version compatible with symfony 5 and even improved from the original and so it resulted in UserBundle
